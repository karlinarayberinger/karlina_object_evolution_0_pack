RIEMANN_SUM

STEP_0: Select a function, y = f(x), to graph on a two-dimensional Cartesian grid between x = A and x = B.

#f_menu_container

STEP_1: Select the number, N, of equally-sized partitions of the x-axis between x = A and x = B to produce.

N rectangles will be drawn inside of each of those N partitions in order to approximate the area whose bounds are y = f(x), x = A, x = B, and y = 0.

#p_menu_container

STEP_2: Select whether to use the left rule, the right rule, or else the midpoint rule.

If the left rule is selected, then each rectangle's left side will overlap with the leftmost point of its respective x-axis partition.

If the right rule is selected, then each rectangle's right side will overlap with the rightmost point of its respective x-axis partition.

If the midpoint rule is selected, then each rectangle's vertical line of symmetry will overlap with the middle point of its respective x-axis partition.

#r_menu_container

STEP_3: Click the GENERATE button to graph y = f(x) between x = A and x = B, to draw the N equal-width rectangles, to compute each rectangle's area, to compute the sum of those areas, and to compute the actual area of the region bound by y = f(x), x = A, x = B, and y = 0.

STEP_4: Click the RESET button to return the web page interface to its initial state.
