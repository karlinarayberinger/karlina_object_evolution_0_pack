PROBABILITY

STEP_0: For each unique COLOR value, select a nonnegative integer number of times to include that COLOR value as an element of an array named A.

#c_menus_div

STEP_1: Specify whether to randomly remove elements from A until A is empty ("probability without replacement") or else to randomy remove elements from A while replacing each of those elements with exactly one newly randomly generated COLOR value from the list of all possible colors enumerated in STEP_0 for a total of N selections ("probability with replacement").

Note that N is the total number of elements in the array named A.

#p_menu_div

STEP_2: Click the GENERATE button to create an array named A whose elements are the COLOR values which each appear as many times as its corresponding input quantity in STEP_0 specifies, arranges the elements of A in a random order, and then allows the user to select one element of A at random to remove from A until A is empty or, if the "probability with replacement" option was selected in STEP_1, until N elements were selected at random and removed from A.
